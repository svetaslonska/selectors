This page demonstrates how CSS selectors work.

id

ids can be applied to only one item on a page

Here's how we hook an id attribute to an element:

id="id-div"

Here's how the CSS is applied to the element:

div#id-div{
    background-color:yellow;
}

class

ids can be applied to only one item on a page

Class can be apply more than one item on a page

class="my-list"

compound

A compound selector allows multiple selectors separated by commas. In this way we can apply the same rules to several selector.

Here's how the CSS is applied to the element:

div#new-div,li.new-list{
     background-color:blue;
}

Here's what it looks like:

I'm inside a div with an id of new-div
  • I'm in an li with a class named new-list

child

A child selector allows us to access next imidiate elements inside the element.

Here's how the CSS is applied to the element:

ul.my-kids > li{
     background-color:green;
}

Here's what it looks like:

  • I'm in an li inside a ul with a class of my-kids

descendant

Using a descendant selector we can access all element at any depth within the current element.

Here's how the CSS is applied to the element:

div.grandkids li{
     background-color:red;
}

multiple classes

We can apply multiple classes to an element by placing class names separated by spaces in the class attribute.

Here's how we hook a multiple classes to an element

class="add-border add-background"

Here's how the CSS is applied to the element:

border:1px solid black;
     }
